# selective_answering

This repository contains the `selective_answering` project from the user's
`intern26/intern1/projects/tmp` workspace. It includes configuration files,
training code, data utilities and experiment results.

## Structure

- `ckpt/` – model checkpoints (not tracked, large files should be removed)
- `configs/` – YAML/JSON configuration definitions
- `data/` – dataset utilities or example inputs
- `results/` – output logs and evaluation
- `scripts/` – shell and python helper scripts
- `src/` – source code for training and inference

### Usage
Clone repo and adjust paths/configs as needed before running experiments.
